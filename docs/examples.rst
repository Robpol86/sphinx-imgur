.. _examples:

========
Examples
========

This page shows several example use cases for the extension.

Embedded Albums
===============

With Details
------------

.. code-block:: rst

    .. imgur-album:: hWyW0

.. imgur-album:: hWyW0

Without Details
---------------

.. code-block:: rst

    .. imgur-album:: 9YZHA
        :hide_post_details: True

.. imgur-album:: 9YZHA
    :hide_post_details: True
